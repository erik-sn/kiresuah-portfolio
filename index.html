Portfolio

Below is a list of the public applications I have developed, a description of them and the base technologies used. Most of these were inspired by exercises at Free Code Camp.


Random Quote Generator - jQuery

Retrieves random quotes from designers and searches google for their picture, then displays both.

Local Weather - jQuery

Retrieves your current local weather and hourly forecast and displays it on an environment.

Twitch Tracker - jQuery

Search and add Twitch.TV streams to a feed gives some extra information on any active streams.

Wikipedia Viewer - jQuery

Allows you to search for wikipedia pages or request a random page, displays some initial information about the topic, and then keeps track of which pages you have looked.

React Calculator - React

A simple implementation of a calculator built in JavaScript using React.JS. Use either your mouse or keyboard to run basic evaluations.

Tic Tac Toe - React

Classic Tic-Tac-Toe game. It features an "unbeatable" AI which will draw or win every match, implemented using the minimax algorithm.

Pomodoro - React

A time management application that cycles between working and breaks. The clock timers and face can be adjusted.

Simon - React

Recreation of the 80's simon says game where the player must match the colors the game produces.

Markdown - React/Django

Type in text and have it be converted to Github styles. If you login through Github you can save and retrieve markdowns to continue working on them.
